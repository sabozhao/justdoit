# 智能刷题平台 - 环境配置说明

## 🌐 API域名配置

本项目支持动态切换API域名，可以根据不同环境自动配置后端API地址。

### 📁 环境配置文件

- `.env` - 默认环境配置
- `.env.development` - 开发环境配置  
- `.env.production` - 生产环境配置

### 🔧 配置内容

#### 开发环境 (.env.development)
```bash
VITE_API_BASE_URL=http://localhost:3005/api
VITE_APP_ENV=development
```

#### 生产环境 (.env.production)
```bash
VITE_API_BASE_URL=https://examtest.top/api
VITE_APP_ENV=production
```

### 🚀 启动方式

#### 1. 使用启动脚本（推荐）
```bash
# 本地开发模式
./start.sh local

# 生产环境模式
./start.sh prod
```

#### 2. 使用npm脚本
```bash
# 本地开发模式
npm run dev:local

# 生产环境模式  
npm run dev:prod

# 默认开发模式
npm run dev
```

#### 3. 构建不同环境
```bash
# 构建生产环境
npm run build

# 构建开发环境
npm run build:dev
```

### 🔍 环境检测

在浏览器控制台中，开发环境会显示：
```
🌐 API Base URL: http://localhost:3005/api
🔧 Environment: development
```

### 📝 注意事项

1. **开发环境**: API指向 `http://localhost:3005/api`
2. **生产环境**: API指向 `https://examtest.top/api`
3. 确保对应的后端服务正在运行
4. 生产环境需要确保CORS配置正确

### 🛠️ 自定义配置

如需修改API地址，请编辑对应的环境配置文件：
- 修改 `.env.development` 中的 `VITE_API_BASE_URL`
- 修改 `.env.production` 中的 `VITE_API_BASE_URL`
